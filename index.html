This page demonstrates how CSS selectors work.

Id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Class

Classes can be applied to several items on a page

Here's how we hook an class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

Compound

Compound selectors allow us to use more than one selector for a set of rules. The selectors must be separated by commas

Here's how the CSS is applied to the element:

div#new-list, li.new-list{
              background-color: blue;
            }

Child

The child selector allows us to access the element that is immediately inside the current element

Here's how the CSS is applied to the element:

ul.my-kids > li{
            background-color: green;
          }

Here's what it looks like:

  • I am inside an li of a child of ul with class of my-kids
  • I am inside an li of a child of ul with class of my-kids
  • I am inside an li of a child of ul with class of my-kids
Descendant

The descendant selector allows us to hook to an element at any depth inside the current element

Here's how the CSS is applied to the element:

div.grandkids li{
          background-color: red;
        }

Here's what it looks like:

  • I am inside an li of a descendant of a div with a class of grandkids.
  • I am inside an li of a descendant of a div with a class of grandkids.
  • I am inside an li of a descendant of a div with a class of grandkids.
Multiple

We can apply multiple classes by placing them in the class attribute with a space between each

Here's how we hook multiple classes to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:


.add-border{
  border: 1px solid #000;
}
.add-background{
  background-color: gold;
}
